1. Название проекта

LaboratorGit – интерфейс авторизации в современном стиле LeetCode

2. Краткое описание

Проект представляет собой экран авторизации, выполненный в минималистичном и современном стиле. Левая часть интерфейса содержит мотивационный слоган «Improve your mind solving problems» на глубоким синем градиенте. Правая часть включает поля Email и Password, кнопку Sign In, ссылки Sign Up и Forgot password?, а также кнопки входа через Google, Facebook и X.
Дизайн создан для демонстрации навыков UI/UX и работы с GitHub.

3. Цель работы / задачи

Освоить основы работы с Git и GitHub.

Научиться создавать и структурировать UI-проекты.

Отработать ведение репозитория: коммиты, ветки, Pull Requests.

Закрепить навыки организации документации через README.md.

Продемонстрировать работу с Figma и экспорт интерфейсов.

4. Инструкции по использованию файлов

/design/ – экспортированные файлы из Figma (если добавлены).
