# LaboratorGit
Это современный экран авторизации в стиле LeetCode. Левая часть содержит мотивационный слоган «Improve your mind solving problems» на темно-синем градиенте. Справа — форма входа: поля для Email и Password, кнопка Sign In, ссылки Sign Up и Forgot password?, а также альтернативный вход через Google, Facebook и X. Дизайн минималистичный, контрастный и ориентированный на удобство пользователя.
